# KrakOFF


Firstly update poetry environment
``` sh
poetry update
```


Run the API in the root project directory:
```sh
poetry run uvicorn app.main:app --reload
```



Build the flutter app in the krakoff directory (flutter directory)

```sh
flutter run
```
